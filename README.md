# Try Ruby!

A command line version of [tryruby.org](http://tryruby.org/) in case that the website goes down or is having troubles with too many simultaneous connections you can use this REPL shell to try ruby!

## Installation

    $ gem install tryruby

## Usage

    $ tryruby
